# Validation Dataset

This directory is reserved for versioned validation materials.

The current repository does not yet include a wet-lab outcome dataset.
For now, validation is limited to:

- automated TypeScript unit tests
- automated Rust unit tests
- TypeScript production builds

Recommended future contents:

- curated OE-PCR design fixtures with expected outputs
- documented external reference comparisons
- experimental gel and sequencing outcomes
- release-tagged reproducibility manifests
